=======================================================
JSON Formatting Demo
=======================================================

[Valid JSON]
Input : {"name": "Alice", "age": 30}
Passed: True
Output: {
  "name": "Alice",
  "age": 30
}

[Markdown fences]
Input : ```json
{"name": "Bob"}
```
Passed: True
Output: {
  "name": "Bob"
}

[Single quotes]
Input : {'name': 'Charlie', 'score': 95}
Passed: True
Output: {
  "name": "Charlie",
  "score": 95
}

[Trailing comma]
Input : {"key": "value",}
Passed: True
Output: {
  "key": "value"
}

[Broken]
Input : This is not JSON at all: ??? {]
Passed: True
Output: {"error": "Invalid JSON after repair attempt", "raw": "This is not JSON at all: ??? {]", "detail": "Expecting value: line 1 column 1 (char 0)"}
